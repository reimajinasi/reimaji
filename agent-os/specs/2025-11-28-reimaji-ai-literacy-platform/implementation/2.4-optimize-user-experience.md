# 2.4 Optimize User Experience

## Ringkasan
- Responsif mobile, loading states, error handling, dan onboarding.

## Implementasi
- Tambah `src/app/loading.tsx` dan `src/app/error.tsx` untuk state global.
- Perkuat responsif dengan grid `md:` di halaman admin baru dan landing.
- Tambah halaman badges `(app)/badges` untuk pencapaian.
- Siapkan basis onboarding melalui halaman dan schema `onboarding` (tracking langkah).

## Acceptance
- UX mulus di mobile/desktop, ada loading dan error recovery.
- Onboarding flow tersedia menuju News, Research, dan LMS.

## Verifikasi
- `npm run type-check`, `npm run lint`, `npm run build` hijau.
- Uji manual: navigasi halaman utama dan `(app)` tanpa error.

## Risiko/Follow-up
- Onboarding langkah detail akan dilanjutkan saat integrasi AI dan konten final.
