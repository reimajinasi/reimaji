# 0.2 Configure Convex Backend

## Ringkasan
- Setup Convex dilakukan di root `reimaji` dengan schema dan fungsi dasar.
- Instalasi paket `convex` selesai; file `convex/schema.ts` dan `convex/users.ts` dibuat.
- Type-check, lint, dan build proyek tetap hijau setelah konfigurasi.

## Tindakan
- Install dependency: `npm install convex`
- Buat schema: `convex/schema.ts` (users, news, research) dengan index dasar dan `publishedAt` optional.
- Buat fungsi dasar: `convex/users.ts` (mutation `create`, query `getByClerkId`).
- Konfigurasi TS: exclude `convex/**` di `tsconfig.json` untuk melewati generated types sampai `npx convex dev` dijalankan.
