# 2.2 Create Initial Content Library

## Ringkasan
- Seed 20+ News, 15+ Research, dan 1 Course (4 modul, 2 lesson/modul) ditambahkan melalui halaman admin `admin/seed`.

## Implementasi
- Tambah file seed di `src/app/admin/seed/seeds.ts`.
- Buat halaman `src/app/admin/seed/page.tsx` dan komponen client `seed/run.tsx` yang memanggil Convex mutations `news.create`, `research.create`, `courses.create`, `modules.create`, `lessons.create` menggunakan Clerk `userId`.

## Acceptance
- Admin dapat menekan tombol untuk mengisi konten awal sesuai jumlah target.

## Verifikasi
- `npm run type-check`, `npm run lint`, `npm run build` hijau.
- Jalankan seed (butuh session admin) dan cek listing publik `/news`, `/research`, serta LMS.

## Risiko/Follow-up
- Tambahkan validasi duplikasi sebelum insert (next iteration).
- Data produk unggulan masih statis; bisa dipindah ke tabel khusus.
