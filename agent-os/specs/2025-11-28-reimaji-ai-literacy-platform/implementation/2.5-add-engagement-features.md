# 2.5 Add Engagement Features

## Ringkasan
- Bookmarking konten, berbagi progres kursus, dan sistem badge.

## Implementasi
- Bookmarks: tabel `bookmarks`, query/mutation `convex/bookmarks.ts`, tombol `BookmarkButton` dipakai di News/Research.
- Share progress: komponen `ShareProgress` ditampilkan di header kursus.
- Badges: tabel `badges` + `user_badges`, query/mutation `convex/achievements.ts`, awarding saat course selesai di `progress.markLessonComplete`.

## Acceptance
- Pengguna bisa bookmark item, membagikan progres, dan melihat badge.

## Verifikasi
- `npm run type-check`, `npm run lint`, `npm run build` hijau.
- Uji manual: bookmark di halaman News/Research, bagikan progres di LMS, badges tampil di `(app)/badges`.

## Risiko/Follow-up
- Tambah UI badge di profile; tambah undo/redo untuk bulk ops.
