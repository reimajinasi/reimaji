# 2.3 Enhance Admin Functionality

## Ringkasan
- Tambah Content Analytics (News/Research + distribusi role) dan User Engagement (aktivitas LMS per course).
- Implementasi Bulk Content Operations untuk publish/unpublish/remove massal.

## Implementasi
- Content analytics: `news.stats`, `research.stats`, `users.roleStats` (Convex) dan halaman `admin/analytics`.
- Engagement: `progress.courseStats` (Convex) dan halaman `admin/engagement`.
- Bulk ops: halaman `admin/bulk` dengan checkbox dan aksi massal untuk News/Research.

## Acceptance
- Admin dapat melihat metrik ringkas konten dan distribusi user.
- Admin dapat melakukan publish/unpublish/remove massal.

## Verifikasi
- `npm run type-check`, `npm run lint`, `npm run build` hijau.
- Uji manual: `/admin/analytics`, `/admin/engagement`, `/admin/bulk` (butuh session admin).

## Risiko/Follow-up
- Tambahkan pagination untuk dataset besar.
- Tambahkan grafik dan tren waktu untuk insights lanjutan.
