# 0.11 Set up Testing Framework

## Ringkasan
- Konfigurasi Jest + Testing Library untuk komponen React.
- Tambah util testing Convex dan data seeds.
- Konfigurasi Playwright untuk E2E dasar.

## Langkah Implementasi
- Tambah devDependencies dan skrip `test*` di `package.json`.
- Buat `jest.config.ts` dan `jest.setup.ts`.
- Buat contoh test `src/components/ui/__tests__/button.test.tsx`.
- Tambah util `tests/convex-test-utils.ts` dan `tests/seeds.ts`.
- Konfigurasi `playwright.config.ts` dan test `tests-e2e/basic.spec.ts`.

## Acceptance
- `npm run test` berjalan dan lulus dasar.
- Struktur test siap dipakai untuk komponen dan util backend.

## Risiko/Follow-up
- Sesuaikan mapping alias jika ada perubahan `tsconfig`.
- Tambah cakupan test untuk fitur News/Research/LMS.
