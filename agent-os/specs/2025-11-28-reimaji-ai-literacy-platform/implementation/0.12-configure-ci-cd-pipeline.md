# 0.12 Configure CI/CD Pipeline

## Ringkasan
- Tambah GitHub Actions untuk lint, type-check, build, dan test.
- Siapkan workflow deploy Vercel bersyarat secrets.

## Langkah Implementasi
- Buat `.github/workflows/ci.yml` untuk pipeline build & test.
- Buat `.github/workflows/deploy.yml` untuk deploy otomatis jika secrets tersedia.

## Acceptance
- Pipeline CI berjalan pada push/PR ke `main`.
- Deploy otomatis aktif saat secrets diisi.

## Risiko/Follow-up
- Pastikan secrets Vercel di repository settings.
- Tambah caching dan matrix Node versi jika diperlukan.
