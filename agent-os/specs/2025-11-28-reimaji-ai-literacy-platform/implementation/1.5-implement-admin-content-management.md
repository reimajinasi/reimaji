# 1.5 Implement Admin Content Management

## Tujuan
- Admin CRUD untuk News & Research.
- AI-assisted summarization (fallback lokal jika API tidak tersedia).

## Implementasi
- Admin News: `src/app/admin/news/page.tsx` (form create, listing; auto-summary via `computeSummary`).
- Admin Research: `src/app/admin/research/page.tsx` (form create, listing; auto-summary via `computeSummary`).
- Convex RBAC: `ensureRole` di `create/update/remove` memastikan hanya admin/superadmin.

## Verifikasi
- `npm run type-check` → OK
- `npm run lint` → OK
- `npm run build` → OK; route `/admin/news` dan `/admin/research` terdeteksi.

## Bukti

- Data yang dibuat tampil di listing publik.
