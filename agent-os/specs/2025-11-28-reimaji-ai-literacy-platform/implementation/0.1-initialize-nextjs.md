# 0.1 Initialize Next.js 16 project with App Router

## Ringkasan
- Inisialisasi Next.js 16 dipindahkan ke root directory `reimaji` sesuai instruksi.
- Tailwind v4 aktif via `@tailwindcss/postcss`.
- ESLint default Next.js, Prettier ditambahkan, Husky + lint-staged untuk Git hooks.

## Tindakan
- Scaffold awal di `web/` untuk validasi cepat, lalu migrasi penuh ke root `reimaji`.
- Setup root: `package.json`, `tsconfig.json`, `eslint.config.mjs`, `next.config.ts`, `postcss.config.mjs`, `.prettierrc`, `.prettierignore`, `.gitignore`.
- Struktur app di root: `src/app/{layout.tsx,page.tsx,globals.css}`, `public/*`.
- Husky init di root: `npx husky init`; `pre-commit` → `npx lint-staged && npm run lint && npm run type-check`.
- Tambah `lint-staged` di root `package.json`; install `prettier`, `husky`, `lint-staged`.
- Tailwind v4 via PostCSS plugin; tambah `tailwindcss-animate`.
