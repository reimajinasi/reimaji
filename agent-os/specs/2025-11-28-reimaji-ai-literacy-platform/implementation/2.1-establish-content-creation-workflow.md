# 2.1 Establish Content Creation Workflow

## Ringkasan
- Template konten untuk News/Research ditambahkan di `src/lib/content/templates.ts`.
- Workflow review/publish/unpublish disiapkan via halaman admin `admin/workflow` dengan kontrol sederhana.

## Implementasi
- Buat `templates.ts` berisi helper `createNewsTemplate` dan `createResearchTemplate` untuk struktur ringkasan 1 paragraf.
- Tambah halaman `src/app/admin/workflow/page.tsx` dan komponen client `workflow-controls.tsx` yang memanggil Convex `listAll` + `update` untuk publish/unpublish.
- Gunakan Clerk `useAuth()` agar RBAC di Convex (`ensureRole`) tetap berjalan.

## Acceptance
- Admin dapat melihat daftar draft/published dan melakukan publish/unpublish.
- Template siap dipakai untuk pembuatan konten konsisten.

## Verifikasi
- `npm run type-check`, `npm run lint`, `npm run build` hijau.
- Navigasi ke `/admin/workflow` (butuh session admin) untuk uji manual.

## Risiko/Follow-up
- Perlu history unpublish jika dibutuhkan (tersedia via `publishedAt` null sebagai draft).
- Tambah status granular (review) jika diperlukan saat skala.
